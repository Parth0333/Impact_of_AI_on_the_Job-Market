Project Overview
This project analyzes the impact of Artificial Intelligence (AI) on the job market using a Kaggle public dataset. The analysis focuses on automation risk, skill demand, and salary trends across industries and job roles from 2024 to 2030.
Using Python-based data analytics and machine learning techniques, the project provides data-driven insights to support workforce planning, reskilling strategies, and policy decision-making.

Objectives
Analyze automation risk across industries and job roles
Identify skills that improve job resilience in an AI-driven economy
Study the relationship between automation probability and salary trends
Segment jobs using clustering techniques
Generate predictive insights using machine learning models

Tools & Technologies
Python
Jupyter Notebook
Pandas & NumPy – Data manipulation
Matplotlib – Data visualization
Scikit-learn – Machine learning & clustering

Key Analysis Performed
Exploratory Data Analysis (EDA)
Descriptive statistics & visualization
Correlation and regression analysis
K-Means clustering for job segmentation
Random Forest model for predictive insights
Feature importance analysis

Dataset Information
Source: Kaggle (Public Dataset)
Type: Simulated workforce data
Key Features:
Job Role
Industry Sector
Automation Probability
Skill Category & Deman
Current & Projected Salary
Employment Trend Indicator

Key Insights
Jobs with higher automation risk show slower salary growth
Digital, analytical, and AI-related skills lead to better job stability
Automation impact varies significantly across industries
Workforce resilience depends more on skills than job titles
